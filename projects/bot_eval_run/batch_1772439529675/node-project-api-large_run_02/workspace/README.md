# Project Structure

This is a simple Node.js project with Express for handling HTTP requests. The project includes modules for projects, tasks, members, and comments.

## Directory Structure

```
src/
├── app.js
├── server.js
├── lib/
│   ├── errors.js
│   └── id.js
└── modules/
    ├── projects/
    │   ├── routes.js
    │   └── service.js
    ├── tasks/
    │   ├── routes.js
    │   └── service.js
    ├── members/
    │   ├── routes.js
    │   └── service.js
    └── comments/
        ├── routes.js
        └── service.js
```
