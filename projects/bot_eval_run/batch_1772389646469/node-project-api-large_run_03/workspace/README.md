# Project Structure

This project is structured as follows:

- `src/app.js`: Main application file.
- `src/server.js`: Server startup logic.
- `src/lib/errors.js`: Error handling utilities.
- `src/lib/id.js`: ID generation utilities.
- `src/modules/projects/routes.js`: Routes for projects.
- `src/modules/projects/service.js`: Service layer for projects.
- `src/modules/tasks/routes.js`: Routes for tasks.
- `src/modules/tasks/service.js`: Service layer for tasks.
- `src/modules/members/routes.js`: Routes for members.
- `src/modules/members/service.js`: Service layer for members.
- `src/modules/comments/routes.js`: Routes for comments.
- `src/modules/comments/service.js`: Service layer for comments.
