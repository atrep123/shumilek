# Task Manager CLI

## Usage

### List tasks
```bash
node dist/cli.js list --data <path>
```

### Add a task
```bash
node dist/cli.js add "Task title" --data <path>
```

### Mark a task as done
```bash
node dist/cli.js done <id> --data <path>
```

### Remove a task
```bash
node dist/cli.js remove <id> --data <path>
```
