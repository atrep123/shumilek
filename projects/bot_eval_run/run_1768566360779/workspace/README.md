# Simple TypeScript ToDo CLI
This project provides a simple CLI tool for managing tasks. It uses TypeScript and Node.js built-in modules.

## Usage
- List tasks: `cli list --data <path>`
- Add task: `cli add <title> --data <path>`
- Mark task as done: `cli done <id> --data <path>`
- Remove task: `cli remove <id> --data <path>`
