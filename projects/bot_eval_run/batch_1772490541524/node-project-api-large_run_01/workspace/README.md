# Node.js Project API

This is a Node.js project that provides an API for managing projects, tasks, members, and comments.

## Features
- Health check endpoint
- CRUD operations for projects
- CRUD operations for members within a project
- CRUD operations for tasks within a project
- CRUD operations for comments within a task

## Dependencies
- Express.js
- Supertest (for testing)

## Setup
1. Clone the repository.
2. Install dependencies using `npm install`.
3. Run tests using `node --test --test-concurrency=1 tests/oracle.test.js`.
