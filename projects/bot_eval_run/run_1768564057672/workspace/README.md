# Mini AI

## Description

This is a simple implementation of a character-level Markov chain generator using Python.

## Installation

Pip install mini_ai

## Usage

### Training the Model

```bash
python -m mini_ai.cli train --input <path> --model-out <path> --order <int>
```

### Generating Text

```bash
python -m mini_ai.cli generate --model <path> --length <int> [--seed <str>] [--random-seed <int>]
```
