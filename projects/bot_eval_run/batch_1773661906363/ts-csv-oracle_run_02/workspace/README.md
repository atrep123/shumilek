# CSV Parser

This is a simple CSV parser written in TypeScript.

## Installation

```bash
git clone https://github.com/yourusername/csv-parser.git
cd csv-parser
tsc
```

## Usage

### Parse CSV File

```bash
node dist/cli.js parse --input path/to/file.csv
```

### Get Stats for CSV File

```bash
node dist/cli.js stats --input path/to/file.csv
```
