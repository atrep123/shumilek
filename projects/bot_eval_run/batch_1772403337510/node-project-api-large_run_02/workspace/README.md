# Project Structure

This project is structured as follows:

- `src/app.js`: Main application file.
- `src/server.js`: Server startup logic.
- `src/modules/projects/`: Routes and service for projects.
- `src/modules/tasks/`: Routes and service for tasks.
- `src/modules/members/`: Routes and service for members.
- `src/modules/comments/`: Routes and service for comments.
- `src/lib/errors.js`: Error handling utilities.
- `src/lib/id.js`: ID generation utilities.
