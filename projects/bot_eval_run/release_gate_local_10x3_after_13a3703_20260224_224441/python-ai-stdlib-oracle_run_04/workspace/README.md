# mini_ai

This is a simple implementation of a character-level Markov chain generator using Python's standard library. The package includes a `MarkovChain` class for training and generating text based on n-grams, as well as a command-line interface for easy usage.

## Installation

To use this package, simply clone the repository and run the following commands:

```bash
python -m mini_ai.cli train --input <path> --model-out <path> --order <int>
python -m mini_ai.cli generate --model <path> --length <int> [--seed <str>] [--random-seed <int>]
```

## Usage

### Training a Model

To train a model, use the `train` subcommand:

```bash
python -m mini_ai.cli train --input path/to/input.txt --model-out path/to/model.json --order 2
```

This will generate a JSON file containing the trained Markov chain model.

### Generating Text

To generate text using a trained model, use the `generate` subcommand:

```bash
python -m mini_ai.cli generate --model path/to/model.json --length 100 --seed "a" --random-seed 42
```

This will output a string of 100 characters generated by the Markov chain model.
