# Project Management API

This is a Node.js monolith for a Project Management API. It includes endpoints for managing projects, tasks, members, and comments. The API is designed to be in-memory and does not use any persistence mechanisms.

## Installation

To install the dependencies, run:

```bash
cnpm install
```

## Running the API

To start the API, run:

```bash
cnpm start
```

## Testing

The API is tested using Supertest. To run the tests, run:

```bash
cnpm test
```
