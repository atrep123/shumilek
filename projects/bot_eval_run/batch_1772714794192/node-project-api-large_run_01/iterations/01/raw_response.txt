 ```\n\n## API Endpoints\n\n- **GET /health**: Check the server status.\n- **POST /projects**: Create a new project.\n- **GET /projects**: List all projects.\n- **GET /projects/:projectId**: Get a /projects/:projectId/tasks/:taskId**: Update the status of a task.\n- **POST /projects/:projectId/tasks/:taskId/comments**: Add a comment to a task.\n- **GET
