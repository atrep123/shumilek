# Todo List CLI

This is a simple command-line interface for managing tasks stored in a JSON file.

## Usage

### Add Task
`add <title>` - Adds a new task with the given title.

### List Tasks
`list --data <path>` - Lists all tasks from the specified data file.

### Mark Task as Done
`done <id> --data <path>` - Marks a task with the given ID as done.

### Remove Task
`remove <id> --data <path>` - Removes a task with the given ID.
